PPT设计思维

发表于 2021-10-09 更新于 2022-02-16

视觉化表达

  1. 阅读
    1. 先认真阅读要表达的内容
  2. 理解
    1. 尝试在自己脑袋里构建阅读的内容信息
  3. 想象
    1. 想象一下如果是自己要用图形化的表达方式，怎么做
  4. 表现
    1. 把视觉化的表达方式展示处理

PPT演示结构

SCQA框架

  1. S 场景
  2. C 冲突
  3. Q 疑问
  4. A 答案

讲究起承转合。

内容精炼

  1. 从数据出发，事件具体描述+数据
  2. 从论点出发，展示论点，讲述论据
  3. 从总结出发，（人物） + 事件 + 意义，对文字进行概括

字体选择

  1. 棱角分明的适合运动风
  2. 方方正正适合商务风
  3. 圆润的适合女性和儿童主题
  4. 宋体这样的适合古典风格。

字体排版

粗细字体搭配或者不同字体搭配，有层次感。

整个PPT选好字体后，就保持一致

配色

主色鲜明，色调一致。

主色选择：

1.跟企业主题颜色一致，例如国信就是红蓝两色
2.或者跟幻灯片内容，例如医疗行业就是蓝绿色，可以通过去百度图库用关键字搜索来看看这个行业的配色是什么样的

配色选择

主色选完了后，可能还需要配色，配色也有好几种方式:

  1. 单色的深浅搭配
  2. 邻近色的搭配
  3. 互补色搭配，可以通过色环来看，对角180度的就是互补色
  4. 邻近色加互补色的方式

颜色最好保持同一个种色调，不要有的深有的浅，而且颜色还不一样，会比较乱

配色通过去网上找下色环工具来辅助
另外，选好颜色后，在PPT里可以放到自定义颜色里，这样以后想换一组，可以切换一组颜色

版式设计

确定版面的基本框架

  1. 留白： 内容多，就留白少一点，内容少，就留白多一些
  2. 一致：内容相似的可以用同一个版式，版式也可以自定义，在母版定义里

让内容更容易被人理解
